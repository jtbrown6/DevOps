## Executing Scripts

```bash
$ which python3 <- Note: location belongs below
#!/usr/bin/python3
print ("Hello World")

```

{% hint style="info" %}
Give current user execution permissions to file:
{% endhint %}

```bash
$ chmod u+x filename.py
$ ./filename.py
```
